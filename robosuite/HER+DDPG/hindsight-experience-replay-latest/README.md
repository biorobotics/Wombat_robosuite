## Files Architecture

```
├── D3_controller
    ├── Inverse kinematics controller files

├── rl_modules
    ├── Contains DDPG algorithm script

├── her_modules
    ├── Contains HER algorithm script
    
├── arguments.py
    ├── Contains the hyperparameters such as no. of epochs, batch size for training, no of runs for demo, learning rate, etc.


```
